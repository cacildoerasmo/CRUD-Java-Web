# CRUD-Java-Web

Este é um CRUD para Web do curso de informática de sistemas. Com este projeto é possível fazer CRUD de duas tabela, Estudante e Turma.

# Tecnologias e Ferramentas usadas
 * JSP
 * Servlets
 * CSS
 * Javascript
 * Hibernate
 * MySQL
